\section{Influence of Damping}

\subsection{Mandatory Cases}

The harmonic analysis has been repeated considering three damping values:
\begin{equation}
    \zeta = 1\%,\quad \zeta = 5\%,\quad \zeta = 10\%
    \label{eq:damping}
\end{equation}

These values have been applied consistently in the harmonic analysis, keeping all other model parameters constant: geometry, material, mesh, boundary conditions, type of excitation and frequency range. Only the damping value has been changed between the three analyses.
